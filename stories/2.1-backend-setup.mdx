# Setting up the zomes

> This guide assumes you are inside a nix develop environment with `hc scaffold` available.

1. Scaffold a new zome pair named `notifications_provider` with:

```bash
hc scaffold zome notifications_provider
```

Select the "Integrity/coordinator zome pair" option, and accept the path that the scaffolding tool offers to scaffold the zomes.

2. Add the `hc_zome_profiles_coordinator` and `hc_zome_profiles_integrity` zomes as dependencies with:

```bash
cargo add -p notifications_provider hc_zome_notifications_provider_coordinator
cargo add -p notifications_provider_integrity hc_zome_notifications_provider_integrity
```

3. Go into the newly scaffolded integrity zome's `lib.rs` (its path may be similar to `dnas/lobby/zomes/integrity/notifications_provider/src/lib.rs`) and **replace its contents with**:

```rust
extern crate hc_zome_notifications_provider_integrity;
```

4. Go into the newly scaffolded coordinator zome's `lib.rs` (its path may be similar to `dnas/lobby/zomes/coordinator/notifications_provider/src/lib.rs`) and **replace its contents with**:

```rust
extern crate hc_zome_notifications_provider_coordinator;
```
